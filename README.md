# Habsieg's LLDB Repository
This is a little project of mine to document and tidy up all of my fictional conlang's words in a text file for ease of use. It uses a command-prompt-based terminal system coded in Lua (because i am most familiar with Lua and nothing else.)


## What to know:
1. this is a database
2. this is a language database
3. this is a constructed-language database
4. this is coded in lua
5. i fear to update the code
6. i also fear to update the database

**You might've also noticed the index.html and script.js files, that's because I've also coded a basic terminal in javascript to allow for quick and easy online read-only access to the database for other people like my friends and the general public.**

If you want to check that out, visit https://hellomyfriend161.github.io/cldb-terminal/

This project will receive more updates down the line.
